EVALUATION GUIDELINES:

1. **Signal Direction is Paramount (Score 5 = correct direction)**: If the strategy produces positive returns where the paper predicts positive returns, the implementation is fundamentally sound. Everything else is calibration.

2. **Expect Differences**: A replication using yfinance data will NEVER exactly match a paper using CRSP/Compustat. Typical differences:
   - Sharpe ratios: 20-50% lower is common due to different universe, survivorship bias in free data
   - Returns: 2-5% lower annually is typical
   - Drawdowns: Can be 5-15% worse due to fewer diversifying assets
   - Turnover: May differ by 20-50% due to different universe size

3. **Scoring Rubric**:
   - **5 (Excellent)**: Signal direction correct, Sharpe within 20% of paper, all qualitative properties match
   - **4 (Good)**: Signal direction correct, Sharpe within 40% of paper, most properties match
   - **3 (Partial)**: Signal direction correct but magnitudes significantly different, or some properties don't match
   - **2 (Poor)**: Signal direction correct but very weak, or major properties don't match
   - **1 (Failed)**: Signal direction wrong, or results are clearly nonsensical

4. **Red Flags to Watch For**:
   - Sharpe > 3.0 for any equity strategy (likely a bug)
   - Negative Sharpe for a well-documented anomaly (likely wrong signal direction)
   - Zero turnover (rebalancing not working)
   - 100% turnover every period (something wrong with sorting)
   - All assets in one portfolio group (sorting broken)
   - Returns exactly match paper (suspiciously perfect - possible data snooping)

5. **Distinguish Code Errors from Data Differences**: A lower Sharpe from yfinance data is expected. A NEGATIVE Sharpe for a momentum strategy is a code error.
