1. FIDELITY TO PAPER: Every formula must match the paper exactly. If the paper uses cumulative returns for momentum, do NOT substitute log returns unless the paper explicitly says so.

3. LOOK-AHEAD PREVENTION: Every function must be designed so that data at time T only depends on data at times <= T (or T-lag as specified).
4. PARAMETER EXTERNALIZATION: Every number that appears in a formula (lookback window, percentile cutoff, number of groups) must come from config.py.
5. TESTABILITY: Each function should be independently testable with synthetic data that has known signal properties.
6. DOCUMENTATION: Every function should reference the specific equation/section/table in the paper it implements.
